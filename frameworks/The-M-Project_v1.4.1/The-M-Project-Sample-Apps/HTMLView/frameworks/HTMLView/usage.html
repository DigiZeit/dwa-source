• Place your HTML Documents in a subdirectory into the frameworks directory. For example HTMLFiles. As for this example this files:
    • usage.html
    • view_1.html
    • view_2.html
    • view_3.html
  • Link these directory to espresso via config.json: 
    "libraries":[
        {"name": "HTMLFiles", "refs": ["*"]}
    ]
  • Add a M.HTMLView to your App 
            usage : M.HTMLView.design({

            url: 'usage.html'
        })
    The url param is the filename.
  • Have fun.
